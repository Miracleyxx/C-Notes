# 											编码规范和注意事项

#### 一、C++头文件包含的名称和顺序

1.google规范中头文件的包含顺序：相关头文件、C系统头文件、C++标准库头文件、其他库头文件、项目的头文件

项目中的头文件都应该作为项目源目录的后代列出，例如， `google-awesome-project/src/base/logging.h` 应包括为

```C++
#include "base/logging.h"
```

- 2.在cpp文件中，头文件包含顺序如下：
  - 1.相关头文
  - 2.一个空行
  - 3.C语言系统头文件（例如带<stdio.h>的头文件）
  - 4.一个空行
  - 4.C++标准库头文件 （例如：<iostream>）
  - 5.一个空行
  - 6.其他第三方库的.h文件

7.当前项目的.h文件
